Installation Guide

1 - Install this Github repository: WebFlex Github Source Code

2 - Go to Manage Extensions: "chrome://extensions" & enable Developer Mode

3 - Press Load Unpack, and select the "WebFlexExtension" folder


4 - You should now have it installed! Go on a new tab and try it out!
Don't forget to press Ctrl-Shift-Z to ENABLE/DISABLE
